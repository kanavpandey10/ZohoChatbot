# ZohoChatbot
Designed a chatbot for Zoho's site 24X7 which helps customers with deciding their plans and suggesting plans based on their usage

**libraries needed**:
openai,
Flask, 
request, 
jsonify, 
send_from_directory,
json,
spacy,
uuid,
psycopg2,
SimpleConnectionPool

For the openAI functioncalling make sure you have a valid API key that has enough number if tokens to test
